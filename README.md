# blinky-board
A project created for sleepover! I followed the amazing heart tutorial and made it my own by turning it into a panda my favorite animal :D

Schematics
---
The first step of the process. This involved using a 555 timer, a clock typically used in computers, 2 resisters, a potentiometer r-variable resistor, a button, a battery, a capacitor, and many many leds. I learned how to wire the set up and it was really cool! I think I've gotten used to the basics of the software! Something difficult for me was setting up the footprints as I had many issues with the number of slots (pads I think they are called?) and had to make sure to type everything in carefully. 


PCB
---
The second, and lengthiest, step of the process. Firstly, in the edge cut layer I created the panda shape using 3 arcs. This took a while as there were lots of gaps and overlaps in the early prototypes - when the solution was actually a whole lot simpler! Then, I had to import all the materials from the schematics in - I didn't have enough LEDs, so I made copies of them. Then, I had to arrange them in this beautiful pattern. Finally, the most diffucult step was connecting all the copper wires. There were just so many connections and certain paths that I had to use. But in the end it looks really cool! After that all I had to do was manually configure the leds by changing their footprints in the PCB editior - - it took a bit of trial and error but it looks like a panda!


3D
---
The last step. Honestly it was so nice to see that my project turned out the way that I intended it to! I'm so glad that I was given such a great tutorial to learn from and I can't wait for the PT.2..? where we build it possibly because I would love to have this in my room!
